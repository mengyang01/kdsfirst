C++的vector容器#

介绍：vector用于动态插入和删除元素，基于数组的数据结构，非常方便，能自动分配空间以及释放空间。

特点：1.连续存储；2.动态变化；3.存储元素多样

注意：需有头文件#include<vector>

用法：

创建vector

std::vector<int> Vector; // 创建一个存储整数的空 vector

std::vector<int> Vector(5); // 创建一个包含 5 个整数的 vector，每个值都为默认值（0）

std::vector<int> Vector(5, 10); // 创建一个包含 5 个整数的 vector，每个值都为 10

std::vector<int> vec; // 默认初始化一个空的 vector

std::vector<int> vec2 = {1, 2, 3, 4}; // 初始化一个包含元素的 vector

添加元素

可以使用 push_back 方法向 vector 中添加元素：
Vector.push_back(7); // 将整数 7 添加到 vector 的末尾

访问元素

可以使用下标操作符 [] 或 at() 方法访问 vector 中的元素：

int x = Vector[0]; // 获取第一个元素

int y = Vector.at(1); // 获取第二个元素

获取大小

可以使用 size() 方法获取 vector 中元素的数量：
int size = Vector.size(); // 获取 vector 中的元素数量

删除元素

可以使用 erase() 方法删除 vector 中的元素：
myVector.erase(Vector.begin() + 2); // 删除第三个元素

清空Vector

可以使用 clear() 方法清空 vector 中的所有元素

Vector.clear(); // 清空 vector
